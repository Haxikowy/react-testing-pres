# Testing in react featuring: Vite, Vitest, React Testing Library

This project is a simple todo-app made to practice testing.

## Prerequisites:

- `node` v14+
- `npm` or `yarn`

## Project startup

1. Install dependencies
```zsh
npm install
```

2. Run dev server
```zsh
npm run dev
```

### Other scripts

- build
```zsh
npm run build
```

- test
```zsh
npm run test
```

- preview (no config provided yet)
```zsh
npm run preview
```
